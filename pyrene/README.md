## Icon Font

Updating the icon font:
```
# yarn updateIconfont [css file] [woff file]
yarn updateIconfont ~/Downloads/pyrene-iconfont-19/style.css  ~/Downloads/pyrene-iconfont-19/fonts/pyrene-iconfont-19.woff
```
